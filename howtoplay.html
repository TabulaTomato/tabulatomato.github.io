How to play

game content

You will have to have a computer, tablet or a phone with internet connection and the game website open during the play. The Game Book and Hints are available on the game website.

game objective

This is an adventure puzzle game in which you take the character of Ziggy Wonder. The game consists of three chapters, each providing for about 2 hours of playtime. The goal is to discover the truth behind the secrets you will encounter during the game and to solve a series of puzzles to get there.

setup

Place the board in the middle of the table
Place the location cards deck face down next to the board
Place the character / object / plant / animal cards deck face down next to the board
Place the map in the map slot on the board
There are 6 location slots on the board
There are 7 character slots on the board
There are 15 object / plant / animal slots on the board
Have a computer / tablet / phone with the website open within reach

the gameplay

The game is cooperative, so you should make all the decisions together.
During the game, whenever you discover a new location, you should put the location card in one of the free slots on the board. You can then take the character/object/plant/animal cards which numbers are written on the location card from the deck and put them in the provided slots on the location card. The game will also sometimes tell you to take a card from the deck. When this happens, put the card in one of the free provided slots on the board.
There are three types of cards:

  • ➤ LOCATIONS, marked with capital letters LOC followed by a number
  • ➤ CHARACTERS, marked with numbers. If the card is marked with , you can immidiately speak to this character. If not, the character is not there, or can't communicate for another reason. You can only talk about this character, but you cannot talk to them.
  • ➤ OBJECTS / PLANTS / ANIMALS, marked with numbers. If the card is marked with , this object / plant / animal can be used immidiately and you can put them in your inventory as soon as you get them. Cards without this symbol can only be talked about and they are to be left on the location they were found at.

Actions you can take during the game are:

  • ➤ EXAMINE locations, characters and objects. You can examine any card you want, but you have to examine the cards marked with
  • ➤ See YOUR OWN OPINION on a character, an object and a location
  • ➤ QUESTION CHARACTERS about other characters, objects and locations
  • ➤ GIVE AN OBJECT to a character
  • ➤ USE AN OBJECT on another object
  • ➤ OPEN A NEW LOCATION
  • ➤ ENTER A CODE in an object when instructed

To do these actions, you will use the Game Book as described bellow.
The game will automatically finish when you've discovered what you were supposed to.
There is no time limit for playing the game, and there are no negative points for asking questions or making wrong combinations of objects in the game. So feel free to do what ever comes to your mind.
If you get stuck, use the hint system. But remember to examine all the cards that you're supposed to, speak to all of the characters you meet in the game and open every new location you were suggested to.
Start the game by exploring the cards you were given in the beginning.

how to use the game book

Each chapter has its game book, which is actually an input field that serves as your tool to examine the cards, communicate with characters, use objects, search for new locations and enter codes.

To EXAMINE a location, character or an object, the corresponding number/letter in the input field. You can do this for any card you like, but you have to do it for the cards with the magnifying glass symbol.

For example, if you want to know more about Ziggy 01, enter '01' in the input field and click 'OK' or press Enter. If you want to explore location Our house LOC01, enter 'loc01' or 'LOC01' in the input field and click 'OK' or press Enter.

To QUESTION CHARACTERS about an another character or an object, or to GIVE AN OBJECT to a character, type their according numbers in the input field separated by a comma, without spaces. Type the number of the character you want to speak to FIRST, and the character / object / plant / animal you want to speak about SECOND. If you want to give an object to a character, type the character's number FIRST, and the object's SECOND. If you ask Ziggy about something, you will hear her thoughts about it.

For example, if you want to hear Ziggy's 01 thoughts about her mother 23, enter '01,23' in the input field and click 'OK' or press Enter.

You can also ask a character about themselves by writing their corresponding number two times in the input field separated by a comma, without spaces.

For example, if you want to hear Ziggy's 01 thoughts about herself, enter '01,01' in the input field and click 'OK' or press Enter.

To USE AN OBJECT on another object, type their corresponding numbers in the input field separated by a comma, without spaces. The order of the numbers isn't relevant.

For example, if you want to give dog cookies mix 05 to Laika 40, enter '05,40' or '40,05' in the input field and click 'OK' or press Enter.

To OPEN A NEW LOCATION, type the according address in the input field. You can find the address of every location by looking at the map, marked with a capital letter and a following number.

For example, if you want to go to the Bus Station, enter 'A32' in the input field and click 'OK' or press Enter.

The game will tell you which objects need to be opened by a code when you examine them. To ENTER A CODE type the object's according nubmer followed by the code in the input field separated by a comma, without spaces.

For example, if you want to open a safe 10 with a 999 code, enter '10,999' in the input field and click 'OK' or press Enter.
