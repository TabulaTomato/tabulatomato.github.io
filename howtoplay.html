How to play

game content

You will have to have a computer, tablet or a phone with internet connection and the game website open during the play. The Digital Game Book and Hints are available on Chapters pages.

game objective

This is a narrative puzzle mystery game in which you take the character of Ziggy Wonder. The game consists of three chapters, each providing about 2 hours of playtime. The goal is to discover the truth behind the secrets you will encounter during the game and to solve a series of puzzles to get there.

setup

Place the map on the table
Place the location cards deck face down next to the map
Place the character / object / plant / animal cards deck face down next to the map
Have a computer / tablet / phone with the website open within reach

the gameplay

The game is cooperative, so you should make all the decisions together.
During the game, whenever you discover a new location, you should put the location card on the table, under the location cards that you've already opened. You can then take the character/object/plant/animal cards from the deck according to the card numbers written on the location card, and put them in their provided slots on the location card. The game will also sometimes tell you to take a card from the deck. When this happens, put the card in a free slot on a location card or somewhere next to it. You can use these cards as any other card.
You can do different types of actions during the game, which will be explained later. To do these actions, you will use the Digital Game Book as described in Actions and Digital Game Book section.
The game will automatically finish when you've discovered what you were supposed to.
There is no time limit for playing the game, and there are no negative points for asking questions or making wrong combinations of objects in the game. So, feel free to do whatever comes to your mind.
If you get stuck, use the hint system. But remember to examine all the cards that you're supposed to, speak to all of the characters you meet in the game and open every new location you were suggested to.
Start the game by exploring the cards you were given in the beginning.

types of cards

There are three types of cards:

1. LOCATIONS, marked with capital letters LOC followed by a number

2. CHARACTERS, marked with numbers. If the card is marked with , you can immediately speak to this character. If not, the character is not there, or can't communicate for another reason. You can only talk about this character, but you cannot talk to them.

3. OBJECTS / PLANTS / ANIMALS, marked with numbers. If the card is marked with , this object / plant / animal can be used immediately. Cards without this symbol canâ€™t be used but only talked about.

actions and digital game book

Each chapter has its Digital Game Book, which is actually an input field that serves as your tool to do the actions in the game. It will look like this:

Actions you can take during the game are:

1. EXAMINE locations, characters and objects. You can examine any card you want, but you have to examine the cards marked with .

To examine a location, character or an object / plant / animal, enter their card number in the input field of the Digital Game Book and click 'OK' or press Enter.

For example, if you want to know more about Ziggy (01), enter '01' in the input field and click 'OK' or press Enter. If you want to explore location Our house LOC01, enter 'loc01' or 'LOC01' in the input field and click 'OK' or press Enter.

2. QUESTION CHARACTERS about other characters, objects / plants / animals and locations.

To question a character, enter their card number followed by the card number of the character, object or location that you want to ask about in the input field of the Digital Game Book, separated by a comma, without spaces, and click 'OK' or press Enter. You can also see Ziggy's opinion on things this way, by entering Ziggy's card number followed by the card number of want to see her opinion about in the input field.

For example, if you want to ask Martha (49) about a key (17), enter '49,17' in the input field and click 'OK' or press Enter. Or, if you want to hear Ziggy's (01) thoughts about her mother (23), enter '01,23' in the input field and click 'OK' or press Enter.

3. GIVE AN OBJECT to a character

To give an object to a character, enter the character's card number followed by the object's card number in the input field of the Digital Game Book, separated by a comma, without spaces, and click 'OK' or press Enter. This is done the same way as the previous action.

For example, if you want to give Frank (25) a book (43), enter '25,43' in the input field and click 'OK' or press Enter.

4. USE AN OBJECT / PLANT / ANIMAL on another object / plant / animal

To use an object on another object, type their cards numbers in the input field separated by a comma, without spaces and click 'OK' or press Enter. The order of the card numbers isn't relevant.

For example, if you want to open the door (36) with a key (29), enter '36,29' or '29,36' in the input field and click 'OK' or press Enter.

5. OPEN A NEW LOCATION

To open a new location, type the according address in the input field. You can find the address of every location by looking at the map, marked with a capital letter and a following number. Sometimes the address of the new location will be a solution to a puzzle and sometimes the game will give you a hint that there is a location that you should go to. Don't forget to open new locations that you were suggested to visit.

For example, if you want to go to the Bus Station, find its address on the map. There is A32 written on the Bus Station on the map, so you should enter 'A32' in the input field and click 'OK' or press Enter.

6. Enter A CODE in an object when instructed

The game will tell you which objects need to be opened by a code when you examine them. To enter a code, type the object's card number followed by the code in the input field separated by a comma, without spaces.

For example, if you want to open a safe 10 with a 999 code, enter '10,999' in the input field and click 'OK' or press Enter.
